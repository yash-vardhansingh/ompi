5.4. VPATH builds

VPATH build are fully supported. For example:

shell$ tar xf pmix-VERSION.tar.bz2
shell$ cd pmix-VERSION
shell$ mkdir build
shell$ cd build
shell$ ../configure [...options...]
<... lots of output ...>
shell$ make -j 4
